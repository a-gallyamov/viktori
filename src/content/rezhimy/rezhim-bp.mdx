# Режим БП (Power Supply)

ViktoRi как программируемый источник питания с регулировкой напряжения и тока в реальном времени.

<Callout type="info">
  Режим БП – дополнительная функция, не входящая в основное меню. Для работы должен быть включён в прошивке (`BP_vc`). Подробнее – в разделе [Прошивка](/docs/proshivka).
</Callout>

## Как активировать

Режим БП не выбирается через меню «Режим работы». Для входа:

1. Перейти на экран датчиков (основной экран с показаниями напряжения и тока)
2. Удержать кнопку ПУСК – режим БП запустится

## Как работает

При запуске ViktoRi:

1. Блокирует модуль защиты (если `GUARDA0` включён)
2. Устанавливает частоту ШИМ для силового модуля
3. Запускает DCDC-преобразователь в режиме заряда
4. Устанавливает начальные значения напряжения и тока из прошивки

После этого устройство работает как стабилизированный блок питания – поддерживает заданное напряжение с ограничением тока.

## Управление

| Действие | Результат |
|----------|-----------|
| Вращение энкодера | Изменение текущего параметра (напряжение или ток) |
| Клик ПУСК | Переключение между регулировкой напряжения и тока |
| Удержание СТОП или ПУСК | Выход из режима БП |

### Экран

| Строка | Содержимое |
|--------|-----------|
| Строка 1 | Реальные значения напряжения (В) и тока (А) с INA226 |
| Строка 2 | Заданные значения: `>12000` `2000` – маркер `>` показывает, какой параметр редактируется |

Маркер `>` перемещается между напряжением и током при клике ПУСК.

## Отличия от режима Заряд

| | Заряд | Режим БП |
|---|-------|----------|
| Назначение | Зарядка аккумулятора | Питание внешней нагрузки |
| Завершение | Автоматическое | Только ручное (удержание СТОП) |
| Регулировка | Предустановленные параметры | В реальном времени энкодером |
| Защита АКБ | Полная (напряжение, ток, время, температура) | Упрощённая |
| Привязка к типу АКБ | Да | Нет |

## Когда использовать

- Тестирование электронных компонентов и модулей
- Калибровка измерительных приборов
- Питание устройств с известным напряжением
- Проверка работоспособности нагрузок

<Callout type="warning">
  Режим БП не имеет автоматических защит заряда аккумулятора (контроль напряжения, тока завершения, времени). Не использовать для зарядки АКБ – для этого есть основные [режимы работы](/docs/rezhimy).
</Callout>

## Настройки (компиляция)

Параметры задаются в `1_User_Setup.h` при компиляции:

| Параметр | По умолчанию | Описание |
|----------|-------------|----------|
| `BP_vc` | включён | Включение режима БП (закомментировать для отключения) |
| `BP_volt` | `12000` | Начальное напряжение (мВ). 12000 = 12,0 В |
| `BP_curr` | `2000` | Начальное ограничение тока (мА). 2000 = 2,0 А |

<Callout type="default">
  После запуска режима напряжение и ток можно менять энкодером – начальные значения используются только при входе.
</Callout>

# ULoger

Принимает данные от зарядных устройств, сохраняет на SD-карту и раздаёт веб-интерфейс с графиками по Wi-Fi. Собирается из готовых модулей без пайки — «бутерброд» из плат.

![ULoger – веб-интерфейс](/images/logger/loger-uklon/Uloger.webp)

Прошивки, обсуждение и поддержка — в [Telegram-канале автора](https://t.me/UklonGroup/9249). Это основной источник информации по логеру.

## Сборка

Компоненты, подключение и сборка логгера описаны в разделе [Сборка логгера](/docs/sborka/logger). Набор модулей одинаковый для всех логгеров.

## Прошивка

Первая прошивка выполняется через [загрузчик](/docs/proshivka/zagruzchik) — загрузить `.bin` файл без компиляции исходного кода.

Скачать актуальную версию: [Telegram](https://t.me/UklonGroup/9249)

Дальнейшие обновления — через OTA (по воздуху) в веб-интерфейсе логгера.

<Callout type="warning">
  При обновлении прошивки обязательно обновить и файлы на SD-карте. Прошивка и файлы SD должны быть одной версии.
</Callout>

## Подготовка SD-карты

1. Отформатировать карту в FAT32 (SDHC, Class 10, до 32 ГБ)
2. Из архива с прошивкой извлечь папку `data` (или `SD`) и скопировать её содержимое в корень карты
3. Вставить карту в модуль

<Callout type="error">
  Без файлов на SD-карте логгер не запустится.
</Callout>

## Совместимость с ЗУ ViktoRi

ULoger поддерживает зарядное устройство ViktoRi. В файле `1_User_Setup.h`:

```c
#define LOGGER 2           // режим для ULoger
#define SERIAL_SPEED 9600  // скорость порта
#define LOGGTIME 5         // период отправки данных, секунды
```

Подключение к ViktoRi — 3 провода:

| Arduino | ESP | Назначение |
|---------|-----|------------|
| 5V | 5V | Питание |
| GND | GND | Земля |
| TX (D1) | RX | Данные |

<Callout type="warning">
  TX от ESP к Arduino подключать не нужно. При прошивке Arduino — отключить провод данных от логгера.
</Callout>

## Wi-Fi

### Первое подключение

При первом запуске логгер создаёт точку доступа:

- Пароль: **12345678**
- IP-адрес: **192.168.4.1**

Подключиться к сети и открыть `192.168.4.1` в браузере.

### Домашний Wi-Fi

В веб-интерфейсе указать SSID и пароль домашней сети. После сохранения и перезагрузки логгер подключится к роутеру.

Настройки Wi-Fi хранятся в файле `Settings/Wifi.ini` на SD-карте. Для сброса — удалить этот файл и перезагрузить логгер.

<Callout type="info">
  Рекомендуется назначить логгеру статический IP, чтобы адрес не менялся после перезагрузки.
</Callout>

### Синхронизация времени

- **NTP** — автоматически при подключении к Wi-Fi с интернетом
- **Браузер** — при открытии веб-интерфейса время синхронизируется с устройства пользователя
- **RTC** — если установлен модуль часов, время сохраняется при выключении питания

## Веб-интерфейс

Интерфейс загружается с SD-карты и работает в любом браузере.

- **Файловый менеджер** — просмотр, скачивание и удаление логов
- **Графики** — отображение данных с автообновлением
- **Настройки** — Wi-Fi, время, параметры логирования
- **Обновление прошивки** — OTA через веб-интерфейс

Логи хранятся в CSV-формате, организованы по папкам (год/месяц). Каждое подключённое устройство получает отдельную папку.
